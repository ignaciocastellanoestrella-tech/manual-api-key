Weather Underground API Key – Setup Guide

Follow the three steps below to locate your Weather Underground API Key and your Station ID, and enter them into WXStation Connect.

1

Access your Weather Underground profile

Go to wunderground.com and log in to your account. Once you are logged in, click on My Profile in the top-right corner of the screen. A dropdown menu will appear — select My Devices.

2

Find your Station ID and navigate to API Keys

You are now on the Member Settings page, under the MY DEVICES tab. Here you will see the list of your registered weather stations.

In the ID column you will find your Station ID. Copy it — you will also need it in WXStation Connect.

To access your API Key, click on the API KEYS tab at the top of the page.

3

Copy or generate your API Key

In the API KEYS tab you will see your API Key (hidden by default). Click Show Key to reveal it, then click Copy to Clipboard and paste it into WXStation Connect.

⚠️ No API Key yet? Accept the Terms of Service and click Regenerate Key to generate a new one.
⚠️ Expired API Key? The expiration date is shown below the key. Accept the Terms of Service and click Regenerate Key to get a fresh one.
⚠️ API Key not working in WXStation Connect? Generate a new key by clicking Regenerate Key. Old keys can sometimes stop working and need to be renewed.

Weather Underground API anahtarınızı ve İstasyon Kimliğinizi bulmak ve bunları WXStation Connect'e girmek için aşağıdaki üç adımı izleyin.

1

Weather Underground profilinize erişin

wunderground.com adresine gidin ve hesabınıza giriş yapın. Giriş yaptıktan sonra ekranın sağ üst köşesindeki My Profile düğmesine tıklayın. Açılır bir menü görünecektir — My Devices seçeneğini seçin.

2

İstasyon Kimliğinizi bulun ve API Keys'e gidin

Şu anda Member Settings sayfasındasınız, MY DEVICES sekmesinde. Burada kayıtlı hava istasyonlarınızın listesini göreceksiniz.

ID sütununda İstasyon Kimliğinizi bulacaksınız. Kopyalayın — WXStation Connect'te de ihtiyaç duyacaksınız.

API anahtarınıza erişmek için API KEYS sekmesine tıklayın.

3

API anahtarınızı kopyalayın veya oluşturun

API KEYS sekmesinde API anahtarınızı bulacaksınız (varsayılan olarak gizlidir). Görmek için Show Key'e tıklayın, ardından WXStation Connect'e yapıştırmak için Copy to Clipboard'a tıklayın.

⚠️ Henüz API anahtarınız yok mu? Hizmet Şartlarını kabul edin ve yeni bir tane oluşturmak için Regenerate Key'e tıklayın.
⚠️ API anahtarınızın süresi doldu mu? Son kullanma tarihi anahtarın altında gösterilir. Yeni bir tane almak için Regenerate Key'e tıklayın.
⚠️ API anahtarı WXStation Connect'te çalışmıyor mu? Regenerate Key'e tıklayarak yeni bir anahtar oluşturun. Eski anahtarlar bazen çalışmayı durdurabileceğinden yenilenmesi gerekir.
